CSS


CSS Kaskádové štýly alebo CSS (skratka z angl. Cascading Style Sheets) je všeobecné rozšírenie (X)HTML. Konzorcium W3C označuje CSS ako jednoduchý mechanizmus na vizuálne formátovanie internetových dokumentov. Štýly umožnili oddeliť štruktúru HTML alebo XHTML od vzhľadu.

Viem, že to zatiaľ nikomu nič nehovorí, preto si vysvetlíme základy CSS. Ako už vieme html robí štruktúru stránky, CSS upravuje výzor danej štruktúri, vysvetlíme si ako to funguje na 3 základnych funkciách, ktoré sa riadia podľa toho čo chcete upravovať.Tieto 3 základné si ukážeme

  • Class-y
  • ID
  • Celé tagy

Class-y

S classami vieme označiť viac vecí a zápis v css musí mať bodku a potom názov class-y



Celé tagy

Celé tagy hovoria samé za seba, pri použítí tohto príkazu sa zmenia všetky tak tie isté tagy na to čo ste im určili a robí sa to tak isto a class-y len tam NESMIETE dať bodku. Pre príklad si ukážeme ako zmeniť všetkým paragrafom farbu, veľkosť a pozíciu

ID

ID je skoro to isté ako class, jediný rozdiel je že class je schopné vziať viac elementov a id iba jeden. Jeho zápis je s #.
